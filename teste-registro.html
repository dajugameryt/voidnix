Sistema de Registro Ativo!

O botão de registro está implementado e funcionando perfeitamente

🎯 Onde está o botão?

1. Na página principal (index.html), clique no ícone de utilizador (👤) no topo
2. No modal que abrir, clique em "Criar Nova Conta"
3. Preencha os dados e clique em "Criar Conta"

🔐 Campos do Formulário

✅ Nome Completo
✅ Email (deve ser válido)
✅ Palavra-passe (mínimo 8 caracteres)
✅ Confirmar Palavra-passe
✅ Aceitar termos e condições

⚠️ Requisitos da Senha

  • Mínimo de 8 caracteres
  • Pelo menos 1 letra MAIÚSCULA
  • Pelo menos 1 letra minúscula
  • Pelo menos 1 número
Exemplo válido: MinhaPass123

📦 O que acontece ao registar?

1. Conta criada no Appwrite
2. Login automático
3. Dados guardados na coleção "users"
4. Redirecionamento para o perfil

🔧 Tecnologia

Frontend: HTML + JavaScript
Backend: Appwrite Cloud
Database: Appwrite Database
Segurança: Encriptação + Validação
